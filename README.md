# Poker Score Tracking – Backend

This is a hobby project built to support game nights with friends.  
The purpose is to track poker scores, determine winners, and keep a history of games and wins over time.

## Tech Stack
- .NET 10  
- MSSQL  
- Entity Framework (Code First)
- Xunit v3 for testing

## Pipeline
CI - github actions, run tests before pushing the code.
